## Abstract.
[![Join our community!](https://img.shields.io/discord/548994743925997570?label=Discord&style=for-the-badge)](https://discord.gg/89kQY2KFQz) ![](https://img.shields.io/github/languages/code-size/Confoederatio/AnalyticalEngine?style=for-the-badge)

> [!NOTE]
> Scriptly-CLI utilises UF-Node, whilst Scriptly-UI utilises UF-Electron. Dynamic script plugin handling is not yet coded in, but is planned in future versions.

**Scriptly** is a dual-purpose lightweight scripting framework for bootstrapping applications that require either UIs or CLIs for maximum performance. These two versions are maintained separately due to frontend differences, and are intricately tied with **Universal Framework** (UF), which functions as the backend software engine for all Confoederatio development. You can find UF (Node) [here](https://github.com/Confoederatio/UniversalFramework).

These repositories are not automatically synced, and automatic feature parity synchronisation is planned for the future.
